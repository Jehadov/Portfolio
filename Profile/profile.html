Quiz Project

I developed a Quiz website using HTML, CSS, Bootstrap, and JavaScript, featuring dynamic content management with session storage for tracking and scoring user responses. The quiz allows users to navigate between questions and retake it as needed.

n this project, I used HTML, CSS, Bootstrap, and JavaScript to create a Quiz website. The website is designed as a single-page application where JavaScript controls the display of content, allowing it to mimic a multi-page experience. I implemented session storage to save user responses and data, which are then compared with the correct answers to calculate the user's score. The quiz allows users to navigate both forward and backward through questions and offers the option to retake the quiz after completion.
